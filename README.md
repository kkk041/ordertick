# OrderTick

OrderTick 是一款面向接单场景的 Electron 桌面计时器，适合陪玩、代练、陪练等按时计费的工作流。它把实时计时、订单台账、历史查询、实际到手统计和收入分析整合到一个桌面应用里。

## 简介

OrderTick 专注解决按时接单时最常见的几件事：开单时快速计时，结束后自动结算，遇到 15 分钟制或老板实付与系统结算不一致时可手动修正实际到手金额，并在历史订单和数据看板中统一展示最终收入结果。

## 主要功能

- 实时开始接单、结束接单与本单时长展示
- 分钟制与 15 分钟制计费规则
- 按比例抽成与按小时固定抽成
- 实际到手金额手动覆盖
- 今日订单台账与历史订单查询
- Excel 导入、导出与模板下载
- 收入分析、趋势图和老板维度统计
- 亮色、暗色、少女模式主题切换

## 技术栈

- Electron
- React 18
- Vite
- Ant Design
- XLSX
